# howSum

Given a target number and a list of numbers, return one array of numbers that sums to the target, or `null` if no such array exists.

Numbers may be reused. Multiple valid answers may exist.

## Examples

- `howSum(7, [5, 3, 4])` returns an array that sums to `7`.
- `howSum(7, [2, 4])` returns `null`.
- `howSum(0, [1, 2])` returns `[]`.
