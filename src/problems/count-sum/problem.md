# countSum

Given a target number and a list of numbers, return how many ordered sequences of numbers sum to the target.

Numbers may be reused. Ordered sequences are distinct, so `[3, 4]` and `[4, 3]` count separately.

## Examples

- `countSum(7, [5, 3, 4])` returns `2`.
- `countSum(0, [1, 2])` returns `1`.
- `countSum(7, [2, 4])` returns `0`.
