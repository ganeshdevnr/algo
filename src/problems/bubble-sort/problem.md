# Bubble Sort

## Description

Given an array of numbers, return a sorted array in ascending order.

The original input array should not be mutated.

## Examples

```ts
bubbleSort([9, 3, 7, 4]); // [3, 4, 7, 9]
bubbleSort([]); // []
bubbleSort([42]); // [42]
```

## Constraints

- The input array may be empty.
- The input array may contain negative numbers, zero, and duplicates.
- Return a new sorted array without mutating the input.
- Expected time complexity: `O(n^2)`.
