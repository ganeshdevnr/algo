# Target Sum

Given an array of numbers and a target number, return how many expressions can be built by assigning either `+` or `-` to each number so the expression evaluates to the target.

The input order is fixed. Each number must receive a sign.

## Examples

- `countTargetSumWays([1, 2, 1], 2)` returns `2`.
- `countTargetSumWays([1, 1, 1, 1, 1], 3)` returns `5`.
- `countTargetSumWays([], 0)` returns `1`.
