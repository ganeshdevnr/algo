# canSum

Given a target number and a list of numbers, return whether the target can be produced by summing any sequence of numbers from the list.

Numbers may be reused.

## Examples

- `canSum(7, [5, 3, 4])` returns `true`.
- `canSum(7, [2, 4])` returns `false`.
- `canSum(0, [1, 2])` returns `true`.
